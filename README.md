> [!NOTE]
> If you own a Github account, **please consider leaving a Star** and forking the Repo! Doing this will help us get more feedback to provide you more and better Features.

## Features

<details>
<summary>Dungeons</summary>

- Leap Alerts
- Rag Axe Alerts
- Pad Timer
- Particle Hider
- Party Finder
- Positional Mesaages
</details>

<details>
<summary>Kuudra</summary>

- Auto Kuudra requeue  
</details>

<details>
<summary>QOL</summary>

- SlayerHelper
- Chat Cleaner
- Auto Rejoin
</details>

<details>
<summary>Commands</summary>

* /vexo - Opens the mod configuration menu.  
* /tyfr - Will leave the Party after the current the Dungeon/Kuudra run
* /&lt;floor or kuudra tier&gt;
    * /f5 -> Floor 5
    * /m7 -> Master Mode 7
    * /t4 -> Fiery Kuudra
</details>
