> [!NOTE]
> If you own a Github account, **please consider leaving a Star** and forkimg the Repo! Doing this will help us get more feedback to provide you more and bezter Features.

## Features

<details>
<summary>Dungeons</summary>

- Leap Alerts  
- Rag Axe Alerts  
- Pad Timer  
</details>

<details>
<summary>Kuudra</summary>

- Auto Kuudra requeue  
</details>

<details>
<summary>QOL</summary>

- SlayerHelper  
- Particle Hider  
- Auto Rejoin  
</details>

And more to come!

<details>
<summary>Commands</summary>

- /vexo - Opens the mod configuration menu.  
- /tyfr - Will leave the Party after the current the Dungeon/Kuudra run
- /<floor / kuudra tier>
    - /f5 -> Floor 5
    - /m7 -> Master Mode 7
    - /t4 -> Fiery Kuudra
</details>
